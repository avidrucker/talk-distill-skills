# Changelog — talk-summarize

## [0.1.0] — 2026-05-26

Initial draft. No script — pure prompt for Claude. Locks in the Casey-
Muratori style (TL;DR + Salient Points + Outline) inherited from
warmed-skills' `talk_summary.md` shape. Style rules emphasize concrete-
over-generic, prose-over-bullets in TL;DR, no LLM-isms, and accurate
representation over editorial commentary.

The two reference examples linked from SKILL.md (the two Fulcro
statecharts talks at avidrucker/fulcro-statecharts-talks) ARE the style
guide — they were written by hand before the skill existed.
